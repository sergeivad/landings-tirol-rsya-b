# Tirol RSYA B Design

**Goal:** Create a local `b` variant of the Tirol RSYA landing with a shortened page structure and one new persuasive block after the second screen.

**Approved Structure:**
- First screen with the main offer, conditions, actions, and gift CTA
- Second screen `Щитовидка под контролем`
- New text-and-benefits block based on the provided scheme
- Footer

**Visual Direction:**
- Reuse the current warm background, soft glassy cards, thin berry borders, and red CTA buttons
- Keep the new block structurally close to the provided scheme, but styled with the landing's existing typography and spacing
- Preserve the same popup GetCourse behavior on both new CTA buttons

**New Block Layout:**
- Large light card with a soft border and subtle background glow
- Introductory text about Алена Гусева and her specialization
- Supporting text about symptoms and the new free group
- Centered prompt `Хотите попробовать?`
- CTA button `Забрать подарки`
- Result headline about what the course can provide
- 2x3 benefit grid with cell borders inspired by the reference table
- Final registration text and CTA button `Хочу быть здоровой!`

**Adaptive Rules:**
- Desktop keeps the text card and table feel
- Tablet reduces spacing and keeps the benefit grid readable
- Mobile stacks all content vertically and converts the benefit table into compact cards while preserving the same message order
